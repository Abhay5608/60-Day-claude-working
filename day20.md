Day 20 – Face Puzzle Game with Webcam Integration 🎯

Built an interactive Face Puzzle Game using HTML, CSS, and JavaScript that transforms a captured webcam image into a playable sliding puzzle.

🚀 Features

📸 Webcam Photo Capture

🧩 Dynamic Puzzle Generation

🎚️ Multiple Difficulty Levels (3×3, 4×4, 5×5)

👆 Drag & Drop Support

📱 Mobile Touch Controls

⏱️ Real-Time Timer

🔢 Move Counter

🏆 Leaderboard using Local Storage

🎨 Responsive and User-Friendly Interface


🛠️ Tech Stack

HTML5

CSS3

JavaScript (ES6)

MediaDevices API

Local Storage


💡 Key Learnings

Working with Browser Camera APIs

Image Processing and Puzzle Grid Generation

Drag-and-Drop Event Handling

Mobile Touch Interaction Design

State Management and Performance Tracking


This project enhanced my understanding of JavaScript game development, browser APIs, and interactive UI design while creating a fun and engaging user experience.
