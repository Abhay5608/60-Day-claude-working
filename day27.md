Day 27 - Prior Authorization Story Simulator

📌 Project Overview

Built an interactive Prior Authorization Story Simulator in HTML to understand how the healthcare prior authorization process works through a story-driven experience. The application follows Rahul's healthcare journey, includes branching choices, educational explanations from Priya, a progress tracker, and key takeaways after completing all chapters.

🚀 Features

Interactive story with 8 chapters

Two decision choices after every scene

Progress bar showing journey completion

Character-based learning (Rahul & Priya)

Educational explanations for every stage

Responsive HTML application

Restart option to explore different dialogue paths

Final summary and learning takeaways


🛠️ Technologies Used

HTML5

CSS3

JavaScript


📚 What I Learned

Prior Authorization workflow in healthcare

Interactive storytelling with JavaScript

State management across multiple story chapters

Dynamic progress tracking

Building responsive educational web applications

Creating engaging user experiences using branching decisions
