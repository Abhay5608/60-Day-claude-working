🏥 Hospital Admission Readiness Simulator

A browser-based simulator that helps healthcare professionals evaluate whether a patient is ready for hospital admission by analyzing documentation, insurance, clinical readiness, prior authorization, and operational workflows.

✨ Features

Patient Information Form

Provider & Physician Details

Diagnosis Selection

Admission Type Selection

Prior Authorization Management

Insurance Verification

Admission Readiness Score

Documentation Validation

Clinical Risk Analysis

Bed Availability Status

Workflow Improvement Suggestions

Governance Snapshot

Final Admission Decision

Multiple Scenario Testing

Responsive Modern UI

Interactive Dashboard


🛠️ Technologies Used

HTML5

CSS3

JavaScript

Chart.js (if included)


🚀 How to Run

1. Download or clone the repository.


2. Open index.html in your browser.


3. Enter provider and physician details.


4. Select diagnosis and admission type.


5. Configure Prior Authorization status.


6. Review readiness score.


7. Complete workflow actions.


8. Analyze the final admission decision.
