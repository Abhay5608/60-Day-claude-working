# Day 30 – Supply Chain Optimizer Simulator

## Overview
Built a React-based Supply Chain Optimizer Simulator to explore how business decisions affect supply chain performance.

## Features
- Random company generation
- Supplier, factory, warehouse, transportation & inventory decisions
- Live KPI dashboard
- Supply Chain Score
- Risk & recommendation analysis

## Tech Stack
- React
- HTML
- CSS
- JavaScript

## Learning
- Learned supply chain optimization concepts.
- Understood decision-making trade-offs.
- Improved React application development skills.
