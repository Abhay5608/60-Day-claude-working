Day 31: AI Supply Chain Control Tower Simulator

GitHub Repository Description

Built an AI Supply Chain Control Tower Simulator that replicates real-world supply chain operations by generating operational alerts and requiring users to make data-driven decisions. The simulator emphasizes incident prioritization, corrective actions, KPI monitoring, and operational performance evaluation in an interactive dashboard.

Key Features

Real-time operational alert simulation

Incident prioritization based on urgency

Multiple alert management

Corrective action decision engine

Live KPI tracking after every decision

Performance dashboard with final score

Replay mode for continuous improvement

Responsive HTML interface
