Day 32 – Marketing Strategy Simulator

📌 Project Overview

Built an interactive Marketing Strategy Simulator that helps users create and test marketing plans for businesses and personal brands. The simulator guides users through audience identification, platform selection, content planning, campaign execution, and performance analysis in a realistic workflow.

🚀 Key Features

Business, Personal Brand, and Random Client modes

Target audience analysis

Marketing platform selection

Content pillar planning

AI-generated 30-day marketing roadmap

Randomized marketing events and decision-making

Growth report with performance insights

Reusable prompts for future marketing campaigns

Responsive HTML interface


🛠️ Technologies Used

HTML5

CSS3

JavaScript


📚 What I Learned

Marketing strategy fundamentals

Audience segmentation

Content marketing planning

Platform-specific campaign selection

Growth tracking and reporting

Building interactive web applications using HTML, CSS, and JavaScript
