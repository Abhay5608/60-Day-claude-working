Day 33 – Media Integrity Analyzer

📌 Project Overview

The Media Integrity Analyzer is an AI-powered web application that helps users identify misleading headlines, emotional manipulation, and media bias. It combines headline analysis, emotion detection, manipulation pattern recognition, and a live integrity dashboard to improve media literacy through interactive challenges.
