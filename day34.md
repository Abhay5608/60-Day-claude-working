Day 34 – Marketing Detective Simulator

📌 Project Overview

Built an interactive Marketing Detective Simulator that lets users investigate marketing campaign failures by analyzing evidence, organizing clues, and identifying the root cause behind unsuccessful campaigns. The simulator uses case-based learning to improve marketing analysis and decision-making skills.

🚀 Features

Interactive marketing investigation board

Campaign evidence analysis

Drag-and-drop evidence cards

Root cause identification

Expert verdict and explanation

Marketing learning report

Multiple randomized case scenarios

Responsive HTML interface

Customizable color themes


🛠️ Technologies Used

HTML5

CSS3

JavaScript


🎯 Learning Outcomes

Improved marketing analysis skills

Learned root cause investigation techniques

Practiced evidence-based decision making

Understood common marketing campaign mistakes

Enhanced problem-solving through interactive simulations
