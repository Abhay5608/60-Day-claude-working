Day 39 – PDF Splitter & Merger

📌 Project Overview

A simple web-based PDF Splitter & Merger built using HTML, CSS, and JavaScript. The application allows users to split PDF files into individual pages or selected page ranges and merge multiple PDF files into a single document. Everything runs locally in the browser for a fast and privacy-friendly experience.

✨ Features

Split PDFs by page or page range

Merge multiple PDF files

Drag & drop file upload

PDF preview before processing

Download processed PDF instantly

Responsive and user-friendly interface

Fully client-side (no server required)


🛠️ Tech Stack

HTML5

CSS3

JavaScript (ES6)

PDF-Lib

🚀 How to Run

1. Download the project files.


2. Open index.html in any modern browser.


3. Upload PDF(s).


4. Split or merge the PDF.


5. Preview and download the processed file.
