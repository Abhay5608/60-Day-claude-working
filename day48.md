# Day 48 – The Verdict Engine

## Overview
Developed **The Verdict Engine**, an AI-powered HTML application that evaluates multiple options using weighted decision criteria. The application ranks alternatives in real time, provides transparent source references, and explains how the final verdict was reached through an interactive research workflow.

## What I Completed
- Reviewed the provided learning resources.
- Watched the solution walkthrough.
- Generated the complete HTML application using Claude.
- Saved and tested the application locally.
- Adjusted decision criteria weights and observed live ranking updates.
- Reviewed the sources panel for every data point.
- Explored the **"How this was researched"** panel and analyzed conflict resolution.
- Captured screenshots of the working application.
- Uploaded the HTML file, screenshots, and documentation to GitHub.

## Key Features
- AI-powered decision evaluation
- Dynamic weighted scoring system
- Live ranking updates
- Transparent source attribution
- Research methodology panel
- Conflict detection and resolution
- Responsive, browser-ready interface
